# 图销冠 TopSeller Studio — 产品需求文档

> 状态：草稿 v0.3 | 最后更新：2025-02-15
> 本文档不涉及技术实现细节，仅描述产品功能与市场定位。

---

## 一、产品定位

**一句话定义**：面向独立站站长的 AI 品牌内容工厂——将专业摄影师的品牌视觉经验产品化，让没有品牌经验的团队一上来就能产出专业水准的电商图片、短视频和社交媒体素材。

**核心价值主张**：不只是"省钱生成图片"，而是**压缩品牌视觉的试错周期**。传统路径下，一个新品牌要花几个月摸索自己的视觉语言——找摄影师、反复拍摄、磨合风格。TopSeller Studio 把大牌级的摄影审美和构图经验内置到产品里，让用户第一次出图就能达到专业水准。

**不是什么**：
- 不是通用 AI 绘图工具（不做艺术创作、插画）
- 不是修图软件（不替代 Photoshop 的精细操作）
- 不是单张出图的玩具（核心价值在套图、流水线、品牌一致性）

## 二、目标客户

### 首要客户：独立站站长

两类人群，一个产品入口：

**A. 中国跨境电商团队**
- 在 Shopify / WooCommerce 上经营面向欧美市场的独立站
- 可能是工厂转型自建品牌——有供应链和产品管理能力，但缺乏品牌视觉经验
- 也可能是成熟电商团队从平台（亚马逊/速卖通）转独立站——懂运营和投放，但品牌内容能力是短板
- 团队规模不一定小，但品牌建设方面是新手
- 使用中文工具，生成面向欧美消费者的英文市场内容

**B. 欧美本土独立站卖家**
- 1-5 人小团队或个人创业者
- 自己选品、自己投放、自己做内容
- 同时运营 Instagram / TikTok / Pinterest / Facebook 广告
- 没有专职内容团队，创始人或运营兼任内容生产

### 远期客户：成长中的品牌商

- 已经度过冷启动期，开始关注品牌一致性
- 有品牌调性要求，所有内容需要视觉统一
- 新品上线节奏快，内容产能跟不上投放需求

## 三、市场痛点

### 痛点 1：品牌视觉冷启动成本极高

工厂转品牌、电商转独立站的团队，最大的困境不是没钱拍照，而是**不知道"对的品牌感觉"长什么样**。传统路径：找摄影师试拍 → 不满意 → 换摄影师 → 再试 → 慢慢磨出品牌调性。这个过程耗时数月，花费数万，且结果高度依赖摄影师个人审美。

**产品解决方式**：将专业大牌摄影师的构图、光影、场景营造经验内置为风格模板。用户无需懂摄影，选择一个风格方向，系统自动产出符合该风格的全套产品图。**品牌视觉从"试错摸索"变成"选择题"。**

### 痛点 2：内容产能瓶颈

一个新品上线，需要：白底主图、模特上身图、场景图、细节图、短视频、社交媒体帖子素材、广告图。传统方式需要摄影师 + 模特 + 后期 + 视频团队协作，一个 SKU 完整内容制作周期 1-2 周。但投放窗口和市场节奏不等人。

**产品解决方式**：一张产品实拍 → 几分钟内自动生成全套内容，将内容生产周期从周级压缩到分钟级。

### 痛点 3：多渠道素材适配

同一个产品需要在不同渠道以不同形态出现：Instagram 要 1:1 正方形、TikTok/Reels 要 9:16 竖版、独立站 hero banner 要 16:9 横版、Pinterest 要 2:3。每个渠道还有自己的内容风格偏好。手动为每个渠道单独制作素材，工作量成倍增长。

**产品解决方式**：一次生成，自动适配多渠道格式，同一内容自动裁切/重构为各平台所需规格。

### 痛点 4：品牌一致性难维护

随着 SKU 增多和内容批次增加，"看起来不像同一个品牌"的问题越来越严重。传统流程靠 brand guideline 文档 + 人工审核来保证，效率低且容易走样。AI 生图更容易出现风格漂移。

**产品解决方式**：品牌套件（Brand Kit）机制——固定品牌色系、调性关键词、指定模特形象、logo 规则，每次生成自动注入，确保跨 SKU、跨批次的视觉统一。

### 痛点 5：从静图到视频的断裂

目前行业里 AI 图片生成和 AI 视频生成是两个割裂的工具。用户在 A 工具生成图片，下载，再上传到 B 工具生成视频，中间有格式转换、质量损失、上下文丢失的问题。

**产品解决方式**：生图和生视频在同一个工作流内完成。套图中选定满意的图片作为首帧，直接衔接视频生成模型，数据在工具内流转，用户不需要跳出。

## 四、产品功能模块

### 模块 1：套图工作台（已实现）

**功能描述**：上传一张产品实拍图，自动规划并生成一整套电商所需图片。

- 支持的图片类型：模特上身图、平铺图、细节图、白底图、自定义场景图
- 用户上传产品图（必选）和模特参考图（可选）
- 系统根据槽位类型自动决定哪些图片参与生成（如：模特槽位用产品图+模特图，白底槽位只用产品图）
- 支持单个槽位重跑和全套一键生成
- 对话式微调：对生成结果不满意可通过自然语言描述修改
- 局部精修：蒙版编辑器支持指定区域重绘

**解决的痛点**：#1 品牌视觉冷启动、#2 内容产能

### 模块 2：品牌套件（Brand Kit）（规划中）

**功能描述**：为品牌建立视觉身份档案，所有内容生成自动遵循品牌规范。

- 品牌色系定义（主色、辅色、禁用色）
- 品牌调性关键词（如："极简"、"户外运动感"、"轻奢质感"）
- 固定模特形象库：绑定模特参考图，所有模特图保持人物一致
- Logo / 水印规则：位置、大小、透明度
- 品牌模板：预设好的套图风格组合，新品直接套用

**解决的痛点**：#4 品牌一致性、#1 品牌视觉冷启动

### 模块 3：AI 视频生成（规划中）

**功能描述**：将套图中的静态图片转化为短视频素材。

- 从套图结果中选择满意的图片作为视频首帧
- 接入视频生成模型，生成 3-5 秒产品展示短视频
- 支持多种视频风格：产品旋转展示、模特走动、场景动态化
- 视频结果可进一步通过自然语言描述调整

**解决的痛点**：#5 静图到视频的断裂、#2 内容产能

### 模块 4：多渠道格式适配（规划中）

**功能描述**：同一内容自动输出为各社交媒体和电商平台所需的格式。

- 预设渠道模板：TikTok (9:16)、Instagram Feed (1:1)、Instagram Story (9:16)、Pinterest (2:3)、Facebook 广告 (多种)、独立站 Banner (16:9)
- 智能裁切与重构：不只是裁剪，而是根据画面内容智能调整构图
- 批量导出：一次生成，按渠道打包下载

**解决的痛点**：#3 多渠道适配

### 模块 5：社交媒体分发（远期规划）

**功能描述**：生成的内容一键发布到各社交媒体平台。

- 连接 Instagram、TikTok、Pinterest、Facebook 等平台账号
- 排期发布：设定发布时间，自动推送
- 内容日历：可视化管理各渠道内容排期

**解决的痛点**：#3 多渠道适配（最后一公里）

## 五、核心内容生产流水线

```
产品实拍图
    ↓
[套图工作台] → 模特图 / 场景图 / 白底图 / 平铺图 / 细节图
    ↓
[对话微调 + 蒙版精修] → 满意的成品图
    ↓
[选择首帧 → AI 视频生成] → 3-5s 产品短视频
    ↓
[多渠道格式适配] → 9:16 / 1:1 / 16:9 / 2:3 各规格素材
    ↓
[社交媒体分发] → Instagram / TikTok / Pinterest / Facebook / 独立站
```

每一步的输出是下一步的输入，数据在工具内流转，用户全程不需要跳出。

## 六、竞争格局

| 竞品 | 定位 | 我们的差异 |
|------|------|-----------|
| Canva | 通用设计工具，功能全但不垂直于电商 | 我们专注电商内容生产，workflow 为电商场景深度定制 |
| 美间 / Krea | AI 单张出图 | 我们做的是套图 + 视频 + 分发的完整流水线，不是单点工具 |
| Creative Force | 企业级内容生产管理 | 定价高、面向大企业；我们面向独立站和成长期品牌，更轻量 |
| Pixelcut | 电商图片编辑 | 侧重后期处理，我们侧重从零生成完整内容集 |

**核心差异化**：不是做"AI 生图"这个点，而是做"产品实拍 → 全渠道内容"这条线。单点工具容易被替代，流水线的衔接效率才是壁垒。更深层的差异是：我们将专业品牌摄影经验产品化，降低品牌视觉的准入门槛。

## 七、市场切入策略（第一轮讨论结论）

**种子用户**：独立站站长（Shopify / WooCommerce），优先获取，决策链短，自助注册即可开始使用。

**地域市场**：
1. 中国跨境电商团队（中文 UI，生成欧美市场内容）——最先触达，语言和文化无障碍
2. 欧美本土独立站卖家——需要英文版产品，是规模化增长的方向

**用户第一个 aha moment**：上传一张手机拍的产品图 → 一键生成 5 张大牌质感的套图。让用户在 30 秒内感受到"原来我的产品也能拍出这种效果"。

**关键洞察**：目标客户不一定是小团队。很多是有生产能力的工厂自建品牌、或有丰富电商经验的团队转独立站。他们有运营能力，缺的是品牌视觉经验。产品的深层价值是把品牌视觉的试错周期从月级压缩到分钟级。

## 八、产品路线图（第二轮讨论结论）

### 优先级排序与底层逻辑

路线图的核心判断：**出图/出视频的质量和意图匹配度是一切的前提。** 功能再多，如果生成结果不符合用户预期，用户不会留下来。

```
Phase 0（当前）  打磨出图质量 — 产品一致性、模特一致性、意图匹配
      ↓
Phase 1         套图 → 视频（Sora 2 + character）— 增长引擎
      ↓
Phase 2         多渠道格式适配（轻量版）— 嵌入现有套图流程
      ↓
Phase 3         Brand Kit — 留存与升级
      ↓
Phase 4         社交媒体分发 — 闭环
```

### Phase 0：打磨出图质量（当前最高优先级）

用户流失的首要原因不是功能缺失，而是**生成结果与预期不符**：

- 产品图被丢掉或变形（已通过套图直连 API 修复）
- 模特一致性不稳定——同一套图里模特脸不一样
- 产品还原度不够——颜色、材质、细节偏离原图

**判断标准**：用户上传一张产品图 + 模特图，生成 5 张套图，至少 4 张无需重跑即可直接使用。达到这个水准才进入 Phase 1。

### Phase 1：视频生成（核心增长引擎）

**为什么视频优先于 Brand Kit**：

1. 获客杠杆——"AI 帮我生成了产品视频"在 TikTok/Reels 上自带传播性，图片没有这个效果
2. 付费意愿——独立站当前最缺的投放素材是短视频，Facebook/TikTok 广告全面偏视频，这是真正的痛
3. 竞争壁垒——图片生成工具遍地都是，图片→视频的一站式流转目前没有成熟产品

**模型选择**：Sora 2，理由：
- 质量层级足够高，匹配"大牌感"的产品定位
- Character 模型可复用套图的模特参考图，天然解决视频中的人物一致性
- 与套图工作台的衔接自然：选择满意的图片作为首帧 → 直接生成视频

### Phase 2：多渠道格式适配（轻量版）

不做独立模块。在现有套图流程里加入目标渠道的尺寸预设：

- 用户创建套图时选择目标渠道（独立站 / Instagram / TikTok / Pinterest）
- 系统自动匹配对应的尺寸比例（16:9 / 1:1 / 9:16 / 2:3）
- 导出时按渠道分组打包

几乎零额外 UI，复用现有的 aspectRatio 机制，覆盖 80% 场景。

### Phase 3：Brand Kit

等用户群成熟后再做。理由：

- 核心客户"还没有品牌经验"，连自己的品牌色系和调性都没想清楚
- 等他们用套图 + 视频跑了几个月，沉淀出自己喜欢的风格方向后，Brand Kit 的需求才会自然浮现
- 到那时 Brand Kit 变成留存和升级工具：帮用户把已经验证的风格固化下来，而不是让他们凭空定义品牌

### Phase 4：社交媒体分发

最后做。依赖前面所有环节成熟后才有价值。

## 九、定价策略（第二轮讨论结论）

### 月订阅 + 用量百分比，不按张计费

**核心原则**：不让用户在每一次生成时做"值不值"的判断。

| 模式 | 问题 |
|------|------|
| 按张计费 | 用户每次点"生成"都在花钱，会反复评估每一次结果的价值，满意度阈值被人为拉高，流失率高 |
| 纯包月不限量 | 成本不可控，大用户会把 API 额度吃光 |

**推荐模式**：类似 Claude / Codex 的 usage% 机制：

- 每月固定价格（如 $20/月）
- 包含一定用量配额，以百分比进度条展示（如"本月已用 43%"）
- 不明确标注"能生成多少张图/多少条视频"，避免用户精算单价
- 用量耗尽后可选择等待下月重置或升级套餐

**用户心理账本**：
> "每月 $20，替代了一次线下拍摄 $500-600 的花费。随便用，不用心疼。"

而不是：
> "又花了 $0.5 生成一张图，这张生得不好，亏了。"
